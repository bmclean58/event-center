This is the website for the Andover Community Center.
The website contains five pages: Home, Events, Menu,
Location, and Contact. Each page contains a navbar
that allows the user to navigate to any of the other
pages. The home page contains a welcome banner, an
image of the ACC, and an about us section. The events
page displays three upcoming live events happening at 
the ACC. The menu pages displays a tables showing the
food/beverage options and prices. The location page
includes a google map to the ACC. Finally, the contact 
page inlcudes a contact form for the user to send a 
message. Flexbox was used for layout and alignment.
